How to use:
1. make csv file and save it in PUT CSV HERE folder:
	- csv must have 2 column and 1 row header
	- Column 1 : From file name (extension optional)
	- Column 2: To file name (extension optional)
2. put all files (which you want to rename) inside PUT FILES HERE:
	- All files inside csv list must exist in this folder!
3. click launcher
4. put delimiter for csv file
5. if file name list in csv contains no extension, you must enter extension in console. ex: ".pdf"
6. enter csv file name
7. done
